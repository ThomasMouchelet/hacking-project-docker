
# Hacking escape game

**folders**
> app : React application

> api : Api-plateform application (REST API)

**Get commandes list**
```sh
make help  
```

**Get started dev**

```sh
make dev env=DEV
```

**Deploy**
```sh
make deploy_init
```

**Pull**
```sh
make deploy_pull
```

**Clear cache**
Don't forget to clear cache on php update code
```sh
make clear
```
